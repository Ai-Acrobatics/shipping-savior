## TL;DR

A **dark-mode glassmorphism SaaS landing page** generated by v0.dev, meant to showcase a conservation platform called "TERRA" (with stale "LUMERA" metadata left over from a v0 rename mid-generation). Single-page scroll, Next.js 16 + React 19 + Tailwind 4, with a rich animation layer (16 distinct motions) combining framer-motion character reveals, CSS keyframes, scroll-linked parallax, and state-driven transitions. About 60% of the bundled shadcn/ui primitives are dead weight (unused in the actual page), and 2 components ship half-finished (the d3 world map and the newsletter form).

**Why it matters for Shipping Savior:** 8 of the 16 animations and 5 of the 9 interactions port cleanly to the planned `v1.1 Investor Demo Sprint` home page. The glass-nav-left pattern (16px radius, backdrop-blur pill) is a strong candidate to replace the current full-height sidebar.

## Animation Highlights (top 8)

1. **A05 — AnimatedText char reveal** (framer-motion). Each character animates `y/opacity/blur` with 30ms stagger. Shortest, most reusable primitive in the template.
2. **A07 — Dashboard 3D scroll-tilt**. `rotateX(0deg → 15deg)` driven by element position in viewport. Apple-product-page feel.
3. **A08 — Dashboard slide-up on mount**. 600ms expo ease from `translateY(200px)` — paired with A07 for the "product reveal" one-two punch.
4. **A06 — Stagger-reveal cascade**. 90ms offset between hero sub-elements (headline → sub → CTA → dashboard), each with 900ms blur-reveal keyframe.
5. **A10 — Animated counter**. IntersectionObserver + setInterval, 60-frame count-up to values like `2.4M`, `99.4%`.
6. **A09 — Logo marquee**. 40s infinite `translateX(0 → -50%)` on a doubled logo strip. Respects `prefers-reduced-motion`.
7. **A12 — Feature stacked-card 3D offset**. 4 cards offset 16px right + 2% smaller per position back. Rotates through on 6s cycle.
8. **A04 — Dynamic word swap**. Every 3s, hero headline swaps one word (e.g., `forests → nature → animals`) with a blur+fade transition that also triggers A05 to replay.

Full catalog: `ANIMATIONS.md` (16 animations).

## Design System Snapshot

- **Palette:** black (`#0B0C0F`) + near-white (`#F2F3F5`) + muted gray (`#A7ABB3`) + pink-purple accents (`#EC4899`, `#A855F7`) + pink-peach gradient on emphasis words
- **Type:** Geist sans body, Georgia serif display, Geist Mono wordmark — heavy type contrast (sans vs serif heading) is a defining choice
- **Radius:** 16px (nav pill), 24px (dashboard mock), 56px (wide pill variant), 1.75rem (shadcn base)
- **Glass system:** 6 variants (nav-wide, nav-left, card, card-inner, button, pill) all sharing `rgba(15, 17, 23, ~0.3)` bg + `backdrop-blur(16-24px)` + `1px rgba(255, 255, 255, ~0.08)` border
- **Motion vocabulary:** 3 custom cubic-beziers + linear for scroll-linked work + Tailwind ease-in-out for micro-fades
